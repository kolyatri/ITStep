# Назва проєкту

Короткий опис проєкту в 1–2 реченнях.

## Мета проєкту
Описати, для чого створено цей проєкт і яку проблему він вирішує.  
Наприклад: навчальний проєкт для відпрацювання роботи з API, базами даних або аналітики.

## Технології
- JavaScript / TypeScript
- Node.js
- Express
- PostgreSQL
- Prisma

## Структура проєкту
src/
├─ controllers/
├─ services/
├─ routes/
├─ utils/
└─ index.ts

## Встановлення та запуск
1. Встановити залежності
npm install

2. Створити файл .env
PORT=3000  
DATABASE_URL=postgresql://...

3. Запустити проєкт
npm run dev

## Функціональність
- Реєстрація та авторизація користувачів
- CRUD-операції
- Валідація даних та обробка помилок

## Навчальні цілі
- Практика проєктування REST API
- Поглиблення знань TypeScript
- Робота з базами даних
